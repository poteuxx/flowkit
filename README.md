# Flowkit

An advanced, centralized tools hub consolidating multiple specialized web utilities into a single, professional, and 100% free ecosystem.

## Project Vision
To provide users with high-end, locally-processed tools (Image editing, Video editing, File conversion, etc.) without the need for multiple accounts or subscriptions.

## Status
- **Current Phase**: Phase 4 (UI/UX System Initialization)
- **Architecture**: Micro-App Plugin System
- **Design**: Premium SaaS / Glassmorphism

## Repository Structure
- `/src/core`: Main application shell, sidebar, and routing.
- `/src/modules`: Individual tool implementations (e.g., `mod-symbols`, `mod-convert`).
- `/src/styles`: Universal design system and CSS tokens.
- `/src/assets`: Icons and branding assets (configurable).
